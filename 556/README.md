## 思路
1. 先判断能否组成新的矩阵，否则返回原来的矩阵
2. 根据输入的r和c（行和列），生成一个矩阵
3. 遍历原来的数组nums，依次插入数据到新矩阵中

总结：(以下为最优解方法)
- 遍历一个数组，可以用
    ```
    for _, v := range nums {
        for _, v2 := range v {
            ...
        }
    }
    ```
- 插入数据的时候
    ```
    res[q][a] = v2
    a++
    if a==c{
        a=0
        q++
    }
    ```
